Allure 風控與數據分析專案

內容：

本專案是一套完整的博弈遊戲風控分析系統，從 API 或 JSON 取得遊戲資料後， 自動完成欄位中文化、Excel 報表產出、追殺局破殺率分析與異常警示， 並透過 pytest 與 Allure 產生可視化測試報告。另提供一鍵執行器， 整合本地 Web 伺服器與 Cloudflare 外網分享，方便即時查看與傳遞報告。 前端另建置 GitHub Pages 靜態儀表板，可在瀏覽器直接上傳 JSON 進行風控分析與圖表展示。

功能：

API / JSON 資料擷取與合併、英文欄位繁中對照、RTP 計算、Excel 匯出、 追殺局（ptk / K / T / B）破殺率統計、5% 風控門檻警示、 Google Sheet 同步、Allure 動態測試報告、單檔 HTML 報告打包、 環境變數安全管理、本機 JSON 匯出、網頁拖放分析、Chart.js 圖表、報告下載。

技術：
Python pandas pytest Allure requests openpyxl gspread Java HTML / CSS / JS Chart.js Git / GitHub GitHub Pages GitHub Actions Cloudflare Tunnel .env
